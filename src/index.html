Practica Docker y NGINX

1ra práctica con Docker y NGINX - Aprendiendo los fundamentos

Objetivo de la práctica

En esta primera práctica, se ha aprendido a crear un contenedor Docker con la imagen de NGINX para servir una página HTML estática.

Pasos realizados

  1. Instalación de Docker.
  2. Creación del archivo Dockerfile.
  3. Configuración de NGINX para servir archivos estáticos.
  4. Mapeo de archivos locales a contenedor Docker.
  5. Verificación de la configuración desde el navegador.

Contenido del proyecto

  • Dockerfile: Archivo con configuración para construir la imagen.
  • docker-compose.yml: Configuración para orquestar el contenedor.
  • index.html: Archivo HTML servido por NGINX.

Detalles de la práctica

Comando Descripción
docker build -t nginx-demo . Crea la imagen Docker con NGINX.
docker-compose up -d Levanta el contenedor en segundo plano.
docker compose down Detiene y elimina los contenedores y redes.
